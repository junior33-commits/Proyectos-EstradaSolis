# Calculadora de Distancias (Euclidiana y Manhattan)

## Descripción
Aplicación web desarrollada en Apache NetBeans que permite calcular
la distancia Euclidiana y la distancia Manhattan entre dos puntos.

## Tecnologías utilizadas
- Java
- JSP
- Apache NetBeans
- HTML

## Funcionalidades
- Cálculo de distancia Euclidiana
- Cálculo de distancia Manhattan
- Interfaz web sencilla

## Contexto académico
Proyecto desarrollado como práctica académica en la asignatura
Aplicaciones Web.
